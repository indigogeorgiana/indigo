Udacity Blog Part One: HTML, CSS and the DOM
How would you describe it to your non-technical friend?

  • HTML - Which means hypertext Mark up Language is a markup language used for creating web pages/websites.

  • CSS - Cascading style sheets are used to format the layout of web pages on the internet. CSS can be used to define text syles, table sizes and other aspects of webpages.

  • DOM - The document object model is an application programming interface that html and xml code is designed within.

    • My analogy, building a website is just like building a house.

    For anyone who has ever built a house they would likely tell you that the above statement is not entirely true, but in principal the basics of building a home are similar.

    Firstly, Think of the DOM our foundation in this analogy, for without a reliable strudy DOM (concrete foundation) construction of a website simply is not possible

    Secondly, a get Web Developer or team of developers depending on the size of your project.

    Thirdly,They will use another key element HTML, yes hyper text markup language its otherwise known as to start building.

    Lastly, CSS which will be our design element, with which crisp and colourful images, fonts and other design elements can be used to make our website/house looks all snazzy.

    What is meant by boxifying design?

    Given that all elements needed to design a webpage can fit into boxes, as web developer's we are able to define and divde these elemnts within the box model. Content, Borders, margins and

    What is the box model?

    • Simply put boxifying design is a method of taking everything needed to build a webpage and dividing these four elements into various boxes.

    • Margin - Clears an area outside the border. The margin is transparent

    • Border - A border that goes around the padding and content.

    • Padding - Clears an area around the content. The padding is transparent

    • Content - The content of the box, where text and images appear.
